# 🎬 Netflix Content Analytics Dashboard

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using SQL Server and Power BI. The objective is to uncover insights related to content distribution, ratings, countries, directors, and release trends through data analysis and interactive visualizations.

---

## Tools & Technologies Used

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- Power BI Desktop
- CSV Dataset
- GitHub

---

## Dataset

Dataset: Netflix Movies and TV Shows Dataset

The dataset contains information about Netflix content, including:

- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

---

## Project Objectives

- Analyze Netflix content distribution.
- Compare Movies and TV Shows.
- Identify top content-producing countries.
- Analyze content ratings.
- Discover release trends over the years.
- Find top directors and genres.
- Build an interactive Power BI dashboard.

---

## SQL Analysis Performed

- Total Content Analysis
- Movies vs TV Shows Analysis
- Top Countries Analysis

- Rating
- Release Year

---

## Key Insights

- Movies make up the majority of Netflix content.
- The United States contributes the highest number of titles.
- Content production increased significantly after 2015.
- TV-MA is one of the most common content ratings.
- Netflix offers content from multiple countries and genres.
